# vn.qdp

### 简介
QDP柜台API接口的Python封装，提供原生C++ API中的全部功能，感谢社区成员魏斌的贡献！

QDP特点：
* 高性能柜台，系统延时显著低于CTP
* 同时支持期货、期货期权、贵金属现货
* 上期所查询持仓返回的今昨仓合并为一条（CTP分两条返回）

### 目录说明
* vnqdpmd: 行情API
* vnqdptd: 交易API
* pyscript: 自动封装脚本
* qdpapi：C++ API文件
* docs：QDP开发文档
